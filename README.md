# 🐍 Jornada Python - Hashtag Programação

Projetos que desenvolvi durante a **Jornada Python**, evento promovido pela Hashtag Programação com foco em aplicação prática de Python para resolver problemas reais.

## 🚀 Projetos

### Aula 1 - Automação de Processos
Automação do processo de cadastro de milhares de produtos em um sistema web
*   **Resultados:** O script executa o trabalho de horas em minutos, podendo ser rodado diariamente ou sob demanda
*   **Tecnologias:** Python, PyAutoGUI e Pandas

### Aula 2 - Análise de Dados
Análise de cancelamento de clientes com gráficos e dashboards em Python para identificar as principais causas de cancelamento e possíveis ações de melhoria
*   **Resultados:** Através de gráficos e dashboards, identificamos causas principais e simulamos ações que reduziriam a taxa de cancelamento de **56% para 18%**
*   **Tecnologias:** Python, Pandas e Plotly

### Aula 3 - Inteligência Artificial
Criação de um modelo de Machine Learning capaz de classificar a nota de crédito de clientes (Ruim, Standard ou Boa)
*   **Resultados:** Modelo treinado e testado com **86% de precisão**, permitindo uma análise de risco automatizada
*   **Tecnologias:** Python, Scikit-Learn e Pandas

### Aula 4 - Criação de Chatbot
Desenvolvimento de uma aplicação de chat em tempo real que funciona tanto no navegador quanto como aplicativo
*   **Resultados:** Implementação de front-end e back-end em Python, permitindo a comunicação instantânea com uma IA
*   **Tecnologias:** Python e Sreamlit

> [!TIP]
> -> Para rodar o projeto do Chatbot, você precisa gerar sua própria **API Key** no site da [OpenAI](https://platform.openai.com), por questões de segurança, minha chave pessoal não está disponível no código.
> Após gerar a sua, apenas adicione o link na parte indicada no arquivo `main.py`

## 📦 Bibliotecas Necessárias

Para rodar os projetos deste repositório, você precisará instalar as seguintes bibliotecas:

```bash
pip install pyautogui pandas plotly scikit-learn streamlit openai
```
